# unemployment-analysis
Analysis of unemployment trends in India using Python. Includes data cleaning, visualization, and statistical insights using Pandas, NumPy, Matplotlib, and Seaborn. The notebook explores patterns in state-wise unemployment and highlights key trends.

Employment & Unemployment Data Analysis – India

This project focuses on analyzing unemployment patterns across India using Python. The notebook performs data cleaning, exploratory data analysis, and creates visualizations to better understand how unemployment varies by state, region (urban/rural), and month.

Objectives

Load and clean the unemployment dataset.

Study employment and unemployment trends over time.

Visualize data using charts for better interpretation.

Identify patterns, fluctuations, and insights.

Present conclusions based on visual evidence.

# Features of the Project

Monthly unemployment trend line charts

State-wise unemployment comparison graphs

Correlation heatmaps

Detailed insights and conclusion sections

Clean and easy-to-read visualizations

# Libraries Used

Pandas – Data cleaning and analysis

NumPy – Numerical processing

Matplotlib – Basic chart plotting

Seaborn – Advanced and aesthetic data visualization

Warnings – To suppress unnecessary warnings

# File Included

employment.ipynb – Main Jupyter Notebook containing full analysis
